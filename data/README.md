# Данные для проекта
В этой папке хранятся исходные CSV-файлы, используемые в ноутбуке.

Файлы:
- `heart_train.csv` — Таблица с тренировочными данными, которая содержит данные пациентов для предсказания риска сердечных приступов.
- `heart_test.csv` — Таблица с тестовыми данными.

Исходные данные можно скачать по ссылке:  
[heart_train.csv](https://drive.google.com/file/d/1_FMg2CTGZ83KIRUqfIUTxvpd7MM9IGnt/view?usp=drive_link)
[heart_test.csv](https://drive.google.com/file/d/1UDXLtS_ZmRjH_bdc8bJ05jaSroZphpk1/view?usp=drive_link)

После скачивания разместите их в эту папку (`/data`).
